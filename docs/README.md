# Docs Index

This directory holds product, architecture, curriculum, and planning documentation for Poker OS.

## Canonical Areas

- [`constitution/`](./constitution/README.md) - product rules and behavioral guardrails
- [`architecture/`](./architecture/README.md) - system structure, boundaries, and implementation seams
- [`roadmaps/`](./roadmaps/README.md) - active planning documents and long-range product direction
- [`content/`](./content/README.md) - drill schema and content-specific reference docs
- [`curriculum/`](./curriculum/README.md) - training taxonomy, node map, and authoring guidance
- [`ai/`](./ai/README.md) - AI coaching architecture notes
- [`systems/`](./systems/README.md) - cross-cutting training system designs
- [`ui/`](./ui/README.md) - product-surface notes and UI sprint artifacts
- [`analysis/`](./analysis/README.md) - reserved for analysis and audit notes

## Notes

- `docs/roadmaps` is the canonical home for roadmap documents.
