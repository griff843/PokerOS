# Roadmaps Docs

Canonical planning area for Poker OS roadmaps, requirements, and gap tracking.

## Files

- `MASTER_ROADMAP.md` - long-range phased roadmap across architecture, content, and coaching depth
- `COACH_EQUIVALENCE_ROADMAP.md` - coach-level product direction
- `COACH_EQUIVALENCE_REQUIREMENTS.md` - capability requirements for coach-equivalent quality
- `COACH_EQUIVALENCE_GAP_TRACKER.md` - current gap tracker and priority order

## Notes

- This directory is the canonical home for roadmap documents.
- Avoid creating new planning docs under `docs/roadmap`.
