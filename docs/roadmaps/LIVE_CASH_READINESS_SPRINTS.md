# Live Cash Readiness Sprints

## Goal Bands

### Band 1: Daily Low-To-Mid Stakes Tool
- Standard: credible everyday study loop for live $1/$2 to $2/$5 style play
- Needed: strong gold-lane depth, real-hand reconstruction, trustworthy follow-up assignment, usable daily workflow

### Band 2: Broad Low-To-High Stakes System
- Standard: credible ongoing tool across wider live environments and tougher opposition
- Needed: multiple adjacent gold lanes, stronger grading depth, broader real-hand transfer, higher content coverage

## Current Position

- Gold live-cash lane is now deep and structured:
  - BTN vs BB SRP river bluff-catch
  - turn-to-river sibling transfer
  - memory-ambiguous reconstruction follow-ups
- Real-hand workflow now exists:
  - import
  - manual reconstruction
  - concept mapping
  - targeted follow-up session launch
- Main blocker is no longer “can the app coach one lane?”
- Main blocker is now “can the app repeat this quality across adjacent lanes and daily use without getting brittle or narrow?”

## Sprint Structure

### Sprint 1: Guided Reconstruction And Session Trust
- Product
  - make manual reconstruction feel guided, not form-heavy
  - surface why a follow-up block is bridge-heavy or exact-match heavy
  - keep uncertainty explicit in review and play surfaces
- Content
  - accept and merge next uncertainty-aware turn-to-river batch
  - reject any content that passes semantics but fails canonical schema
- Exit criteria
  - real hand -> review -> follow-up block feels clean and honest
  - users can tell why the system chose bridge reps

### Sprint 2: Adjacent Lane Expansion
- Product
  - preserve the same real-hand bridge into a second adjacent lane
  - keep follow-up assignment lane-aware
- Content
  - build next sibling lane beyond pure river bluff-catching
  - target turn probe / delayed river decision and threshold carryover
- Exit criteria
  - at least 2 adjacent lane families are strong enough for daily rotation
  - daily use no longer feels trapped in one subtree

### Sprint 3: Real-Hand Prescription Tightening
- Product
  - assign drill blocks from explicit leak categories
  - show “what the hand proved” vs “what remains uncertain”
  - improve summary/review surfaces around transfer
- Content
  - bridge drills specifically for common live-memory leak patterns
  - more blocker-direction and pool-misread repair reps
- Exit criteria
  - real mistakes consistently map to the right next training block
  - user can feel the difference between exact-history and memory-ambiguous coaching

### Sprint 4: Coverage And Repetition Safety
- Product
  - improve rotation so repeated daily use does not feel stale
  - harden content/reporting pipeline further if needed
- Content
  - deepen enough adjacent lanes that practice does not overfit one family
  - maintain gold-lane quality bar across all new batches
- Exit criteria
  - enough trusted reps for ongoing daily use at low-to-mid stakes

## Estimated Readiness

### Daily Low-To-Mid Stakes Tool
- likely after Sprint 2 or Sprint 3 if quality stays high

### Broader Low-To-High Stakes System
- likely after Sprint 5 through Sprint 8, depending on how many adjacent lane families reach the same standard
